Header Parser Microservice Project

This project is from FreeCodeCamp's Back End Certification curriculum. It takes a request object and parses the headers for the user's IP address, language, and OS of their browser and responds with a JSON object of that info.

Example Usage

https://thawing-coast-66449.herokuapp.com/whoami

Example Output

{ "ipaddress": "192.22.3.171", "language": "en-US", "software": "Macintosh; Intel Mac OS X 10_12_6"}
